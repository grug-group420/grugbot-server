# Code of Conduct

## grug rules simple

1. **Be nice** - grug no like mean people
2. **Be helpful** - share knowledge, help newbie
3. **Be patient** - everyone learn at own pace
4. **No hate** - racism, sexism, any -ism = ban

## What grug expect

✅ Constructive feedback
✅ Respectful disagreement  
✅ Helping others learn
✅ Celebrating wins together

❌ Personal attacks
❌ Harassment of any kind
❌ Trolling or inflammatory comments
❌ Discrimination

## Enforcement

Break rules = warning first, then ban.

grug say: "community strong when tribe kind" 🪨
