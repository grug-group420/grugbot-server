> AI-powered CLI assistant with zero dependencies. Simple commands, grug-brained responses.

## 🚀 Quick Start

```bash
# With Bun (recommended)
bunx grugbot420 help

# Or install globally
bun add -g grugbot420
grug help
```

## 💡 Philosophy

Based on [The Grug Brained Developer](https://grugbrain.dev/):

- **Zero dependencies** - No node_modules, instant startup
- **Simple commands** - Do one thing well
- **Fast** - Bun.js runtime, no bloat

## 🛠️ Commands

```bash
grug help          # Show available commands
grug wisdom        # Get grug wisdom
grug joke          # Developer humor
grug project       # Project tips
grug debug         # Debug assistance
```
