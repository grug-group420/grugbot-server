# Contributing to grugbot420

grug appreciate help. grug keep simple.

## 🪨 The Grug Way

1. **No complexity** - If solution need 100 lines, find 10 line solution
2. **No dependencies** - stdlib only, no npm install
3. **Ship fast** - Working code > perfect code

## How to Contribute

### Bug Reports
- Open issue
- Describe what broke
- Show what you expected

### Pull Requests
1. Fork repo
2. Make changes (keep simple!)
3. Test locally with `bun run cli.ts`
4. Submit PR

### Code Style
- Clear variable names
- Comments only when truly needed
- No clever tricks

## First Time?

Good first issues labeled `good first issue`. Start there.
